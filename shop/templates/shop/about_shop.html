Лабораторная работа №16

Создание и настройка Django

Django Проект интернет-магазина · базовый старт

Цели

  • Освоить процесс создания веб-приложения с использованием фреймворка Django.
  • Научить базовой настройке проекта и приложений для дальнейшей разработки интернет-магазина.

Задачи

  1. Установить и настроить окружение для работы с Django.
  2. Создать новый проект Django через командную строку.
  3. Настроить базовые параметры проекта (settings.py: база данных, часовой пояс, язык).
  4. Создать приложение внутри проекта для управления функционалом магазина.
  5. Подключить приложение к проекту и проверить его работоспособность.
  6. Настроить базовый URL-маршрутизатор для приложения.
  7. Проверить запуск сервера разработки и отображение стартовой страницы.

Ход работы

Выполните последовательные шаги, следуя инструкции:

  1. Изучите теоретические сведения — ознакомьтесь с основами Django, структурой проекта и приложения.
  2. Выполните общие и индивидуальные задания в тетради для лабораторных и практических работ или на рабочем компьютере.
  3. Сдайте выполненные задания в Classroom или сдайте тетрадь преподавателю.
  4. Защитите выполненные задания — будьте готовы объяснить код и настройки.

⚙️ Базовые параметры (settings.py)

DATABASES — настройка SQLite (по умолчанию)
TIME_ZONE = 'Europe/Moscow' (или ваш часовой пояс)
LANGUAGE_CODE = 'ru-RU'
INSTALLED_APPS — добавить созданное приложение
1 python manage.py runserver — запуск сервера
2 Проверка http://127.0.0.1:8000/
3 URL-маршруты в urls.py (проект + приложение)
Проект: django-admin startproject myshop
Приложение: python manage.py startapp store
Миграции: python manage.py migrate
